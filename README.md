# La Cevicheria Berlin — Website

Official website for **La Cevicheria**, a Peruvian restaurant located at Dresdener Straße 120, Berlin.

🌐 [cevicheria-berlin.com](https://cevicheria-berlin.com)

![Screenshot](./src/images/cevicheriaReadme.png)

## Tech Stack

- **React** — UI framework
- **React Bootstrap** — component library and layout
- **i18next** — multilingual support (DE / EN / ES)
- **React Router** — client-side routing
- **Deployed on** Hostinger
